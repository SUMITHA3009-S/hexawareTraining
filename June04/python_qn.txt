What is python?
Python is a popular, high-level, interpreted programming language widely known for its simplicity, readability, and versatility.

Advantages of using python?
Easy to Learn and Read: Syntax mirrors natural English. Clean formatting relies on indentation instead of complex brackets.
Massive Library Ecosystem: Includes specialized tools like Pandas for data engineering, TensorFlow for machine learning, and Django for web systems.
High Productivity: Requires fewer lines of code compared to C++ or Java. Accelerates rapid prototyping and corporate deployment cycles.
Cross-Platform Portability: Write code once and run it seamlessly across Windows, macOS, and Linux.
Automatic Memory Management: Utilizes built-in garbage collection and reference counting to optimize memory without manual code.
Strong Community Support: Backed by global open-source developers and major tech giants like Google.
Flexible Integration: Embeds smoothly with other foundational languages such as C, C++, and Java. 

5 use cases where python is used?
1. Artificial Intelligence (AI) and Machine Learning (ML)
2. Data Analytics and Data Science
3. Web Development (Backend)
4. Task Automation and Scripting
5. Web Scraping and Data Extraction
